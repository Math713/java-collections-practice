# Java Collections Practice

Este repositório contém exercícios práticos e exemplos de uso das
principais Collections do Java, com foco em aprendizado contínuo
e preparação para estágio em desenvolvimento backend.

## Conteúdos estudados
- Set
  - HashSet
  - equals e hashCode
- Map
  - HashMap
- Streams API
- Optional
- Exceptions

## Estrutura do projeto
- /set → exemplos relacionados a Set (HashSet)
- /map → exemplos relacionados a Map (HashMap)
- /streamss/ → exemplos com Streams API

## Objetivo
Consolidar fundamentos de Java amplamente utilizados em aplicações
backend e exigidos em processos seletivos para estágio.

## Observação
Este repositório tem finalidade educacional e representa minha evolução
nos estudos de Java.
